What is AJAX?

AJAX stands for Asynchronous JavaScript and XMl. AJAX is not an application or language, it is a new technique for creating better, faster and more interactive web applications with help of XML, HTML, CSS and JavaScript. AJAX just uses a combination of:
1. A browser built-in XMLHttpRequest object (to request data from a web server)
2. JavaScript and HTML DOM (to display or use the data)

AJAX advantages are:
1. Can perform GET and POST requests.
2. It can be used to retrieve data from Text file, HTML, XML or JSON data from a remote server.

AJAX disadvantages are:
1. Security is less in AJAX applications as files are downloaded at user side.
2. JavaSciprt disabled browsers cannot use the application.

What is JSON?

JSON (JavaScript Object Notation) is a lightweight format that is used for data interchanging. It is based on a subset of JavaScript language.

JSON advantages are:
1. JSON is (mostly) native to JavaScript, which means you don't have to do computationally-expensive XSL transformations to extract the data.
2. JSON is light, as it can represent the same data in XML in fewer bytes, making network transmissions and read/writes faster.

JSON disadvantages are:
1. Far less bloated, easier to parse and arguably more human readable (if you space it out properly).
2. Not quite as powerful: not expressive enough to separate attributes from values.

CONCEPT 1: Retrieving from a TEXT file and updating portion of page.

CONCEPT OUTPUT:

CONCEPT CODE:

            function concept1() {
              var xhttp = new XMLHttpRequest();
              xhttp.onreadystatechange = function () {
                if (this.readyState == 4 && this.status == 200) {
                  document.getElementById('concept1').innerHTML = this.responseText;
                }
              };
              xhttp.open('GET', 'Divjot.txt', true);
              xhttp.send();
            }

EXPLANATION:

Using Ajax we can make a request using HTTP to a local file. Further using DOM manipulation to insert the content of that local file on the page. The benefit is that we can request data and receive it as well. This is acheived by using XMLHttpRequest.

CONCEPT 4: JSON.stringify() method

PROGRAM OUTPUT:

         {"x":5,"y":6}

         [3,"false",false]

PROGRAM CODE:

              console.log(JSON.stringify({ x: 5, y: 6 }));

              console.log(JSON.stringify([new Number(3), new String('false'), new Boolean(false)]));

PROGRAM EXPLANATION:

The JSON.stringify() method converts a JavaScript object or value to a JSON string, Additionally replacing values if a replacer function is specified or optionally including only the specified properties if a replacer array is specified. If the values are Boolean, Number, and String objects are converted to the corresponding primitive values during stringification, in accord with the traditional conversion semantics. It can also JSON.stringify() can return undefined when passing in "pure" values like JSON.stringify(function(){}) or JSON.stringify(undefined).
